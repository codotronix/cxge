Codotronix Game Framework User Manual

cxgf.GameObject


cxgf.GameObject.create(gameObjectConfig) Creates and returns a new Game object. To create the new game object it takes a JSON parameter called gameObjectConfig.
  1. x: <Number> X Coordinate
  2. y: <Number> Y Coordinate
  3. speed: <Number> Defaults to 1
  4. speedX: <Number> Defaults to speed
  5. speedY: <Number> Defaults to speed
  6. htmlId: ID of HTML DOM element that represents this object
  7. holdDirectionNTurns: Used for random moves only. It indicates how many turns or ticks to wait before randomizing the direction again. Defaults to 10
<GameObjectInstance>.move(direction) Move the object in the given direction. "direction" can be any one of the following strings
  1. "W" or "LEFT"
  2. "E" or "RIGHT"
  3. "N" or "UP"
  4. "S" or "DOWN"
  5. "RANDOM-4" (Default): Randomly move to any direction among E / W / N / S
  6. "RANDOM-8": Randomly move to any direction among E / W / N / S / NW / NE / SW/ SE (PENDING)
  7. "RANDOM-EW": Randomly move to any direction among E / W (PENDING)
  8. "RANDOM-NS": Randomly move to any direction among N / S (PENDING)
<GameObjectInstance>.getAll(identifier) Returns all the Game Objects that match(es) the provided identifier, e.g. identifier: {type: "ball"}
